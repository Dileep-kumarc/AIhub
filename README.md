# 🤖 AIHub – AI Tools Directory

AIHub is a web platform that brings together various **AI tools in one place**, helping users easily discover and explore tools for different use cases.

---

## 🚀 Features

- 🔍 Browse AI tools in one place
- 🧠 Organized categories for easy navigation
- 🌐 Simple and clean UI
- ⚡ Fast and lightweight web application

---

## 🎯 Purpose

The goal of this project is to:
- Help users discover useful AI tools
- Provide a centralized platform for AI resources
- Practice building real-world web applications

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML
- CSS
- JavaScript
